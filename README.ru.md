# boop

Минималистичная социальная сеть в стиле Reddit на чистом PHP без внешних зависимостей. Никаких Composer или фреймворков — только PHP и MySQL.

## возможности

- **Пользователи** — регистрация, вход, профиль с аватаром (смена по клику)
- **Посты** — создание постов с заголовком и форматированным содержимым
- **Форматирование текста** — `**жирный**`, `*курсив*`, `#теги` (авто-ссылки)
- **Вставка медиа** — загрузка изображений/GIF и вставка прямо в текст через `[img]filename.ext[/img]`
- **Голосование** — лайк/дизлайк с переключением (защита от двойных голосов)
- **Комментарии** — обсуждение постов
- **Поиск** — по заголовку, содержимому или `#тегу`
- **Сортировка ленты** — новые или лучшие
- **Постраничная навигация** — 20 постов на страницу
- **Аватары** — автогенерация SVG-инициалов, возможность загрузки своего
- **MySQL** — реляционная БД, схема в `schema.sql`
- **Автомиграции** — таблицы создаются автоматически, новые фичи добавляют свои миграции
- **Безопасность** — пароли хранятся только в виде bcrypt-хеша, валидация MIME при загрузке
- **Языки** — русский и английский интерфейс, переключение одним кликом (EN/RU в навигации)
- **Тема** — минималистичный оранжево-чёрный дизайн

## требования

- PHP 8.0+
- MySQL 5.7+
- Расширения: `fileinfo`, `mbstring`, `mysqli` (обычно включены по умолчанию)

## быстрый старт

```bash
# 1. Создайте базу данных в MySQL
mysql -u root -e "CREATE DATABASE IF NOT EXISTS boop"

# 2. При необходимости отредактируйте доступы к БД в config.php

# 3. Запустите сервер
cd boop
php -S localhost:8000
```

Откройте `http://localhost:8000` в браузере.

Приложение само создаст все нужные таблицы при первом запуске (см. `migrations.php`).

## структура проекта

```
boop/
├── index.php           # главная / лента постов
├── post.php            # просмотр поста + комментарии
├── create_post.php     # редактор поста с drag-drop загрузкой
├── search.php          # поиск по тексту и тегам
├── profile.php         # профиль + смена аватара
├── vote.php            # обработка голосов
├── upload.php          # AJAX-загрузка файлов
├── login.php           # форма входа
├── register.php        # форма регистрации
├── logout.php          # выход
├── config.php          # константы и настройки
├── functions.php       # JSON-хранилище, форматирование, хелперы
├── auth.php            # логика авторизации
├── header.php          # шапка + навигация
├── footer.php          # подвал
├── migrations.php      # автомиграции (создаёт таблицы, новые фичи добавлять сюда)
├── lang/               # переводы интерфейса (en.php, ru.php)
├── style.css           # оранжево-чёрная тема
├── uploads/            # загруженные файлы (avatars, images)
├── README.md
└── README.ru.md
```
